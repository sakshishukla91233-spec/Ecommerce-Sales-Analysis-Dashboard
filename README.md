# E-commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data to identify key trends, customer behavior, and business performance.

## Tools Used
- Power BI
- SQL
- Excel

## Key Insights
- High-value customers contribute most of the revenue
- Sales vary across regions and categories
- Customer segmentation helps identify key business opportunities

## Features
- KPI metrics (Total Sales, Orders)
- Top customers and region analysis
- Sales trend visualization
- Customer segmentation

## Conclusion
This dashboard helps businesses make data-driven decisions and improve customer retention strategies.
